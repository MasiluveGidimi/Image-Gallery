# Image Gallery Project

## 📸 Overview

This project is a responsive **Image Gallery web application** built using **HTML, CSS, and JavaScript**. The gallery allows users to browse a collection of images, view captions, filter images by category using tags, and open images in a lightbox view for a larger display.

The goal of this project is to demonstrate basic **front-end web development skills**, including layout design, DOM manipulation, and interactive user interfaces.

---

## ✨ Features

* Responsive **grid image gallery**
* Image **captions**
* **Tag filtering** to display images by category
* **Lightbox view** to enlarge images
* Hover **zoom animation effect**
* Clean and simple **user interface**

---

## 🛠️ Technologies Used

* **HTML5** – page structure
* **CSS3** – styling and layout (Grid)
* **JavaScript (Vanilla JS)** – interactivity and filtering

---

## 🖼️ How It Works

* Images are displayed using **CSS Grid**.
* Each image has a **category tag** stored in a `data-category` attribute.
* JavaScript listens for **button clicks** and filters images based on their category.
* Clicking an image opens a **lightbox viewer** for a larger preview.
